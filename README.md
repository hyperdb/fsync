# fsync (file sync)

設定ファイルに指定したソースディレクトリからターゲットディレクトリに対してファイルをコピーします。ターゲットディレクトリにファイルがすでに存在している場合はコピーしません。ターゲットファイルのディレクトリが存在しない場合には無条件に作成します。

なお、コピーするパスの組み合わせは複数設定できます。

```python:config.py
conf = [
    {
        "src_path": r"",
        "dst_path": r"",
    },
    {
        "src_path": r"",
        "dst_path": r"",
    },
]
```
